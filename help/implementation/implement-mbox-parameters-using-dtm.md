# DTM을 사용하여 Adobe Target용 Mbox 매개 변수 구현

## 목표

이 비디오를 모두 시청하면 다음 작업을 수행할 수 있습니다.

* target-global-mbox의 매개 변수 또는 프로필 매개 변수에 정적 이름/값 쌍 매핑
* 데이터 요소의 기본 사항 이해
* target-global mbox의 매개 변수 또는 프로필 매개 변수에 동적 데이터 요소 값 매핑
