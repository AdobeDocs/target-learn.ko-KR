* target-global-mbox의 매개 변수 또는 프로필 매개 변수에 정적 이름/값 쌍을 매핑합니다.
* 데이터 요소의 기본 사항 이해
* target-global mbox의 매개 변수 또는 프로필 매개 변수에 동적 데이터 요소 값 매핑
