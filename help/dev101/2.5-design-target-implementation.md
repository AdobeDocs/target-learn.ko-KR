# Target 구현 디자인

## 설명

이 비디오에서는 다음 방법을 배웁니다.

* 표준 사이트(다중 페이지) 및 단일 페이지 앱(SPA) 환경의 구현 차이 식별
* 프로세스를 적용하여 사용자 기반 캠페인 구현(mbox3rdPartyID)
* 시스템 아키텍처/설계 문서에서 관련 정보 식별
* 구현 의미를 위한 보고 전략 분석

## 대상

* 개발자
* 아키텍트
