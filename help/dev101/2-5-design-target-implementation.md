# [!DNL Target] 구현 디자인

이 비디오에서는 다음 방법을 알아봅니다.

* 표준 사이트(다중 페이지) 및 단일 페이지 앱(SPA) 환경의 구현 차이점 식별
* 사람 기반 캠페인(mbox3rdPartyID)을 구현하는 프로세스 적용
* 시스템 아키텍처/설계 문서에서 관련 정보 확인
* 보고 전략을 분석하여 구현에 미치는 영향

## 의도한 대상

* 개발자
* 설계자
